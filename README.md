# Blade Messenger Lite
Blade Messenger is a futuristic messaging app combining offline SMS and a smart AI assistant called Blade AI. It features a premium dark 3D neon interface, customizable theme colors, and supports 11 languages including Kiswahili, English, Russian, Chinese, Arabic, Turkish, Japanese, Korean, Hindi, and Persian.

## Features
- Offline SMS: Send and receive messages without internet.
- Blade AI: Assist offline or online, answer questions, generate content.
- Multi-Language Interface: Users can switch languages easily.
- 3D Neon UI: Dark background with neon highlights.
- Theme Customization: Adjust accent colors.
- Personalized Greetings: Welcome and goodbye messages.
- Security Minded: Root detection, secure storage, anti-screenshot.
- AI Image Generation: Create images using Blade AI (online mode).

Designed by Tarik, founder of Cipher Tech in Masasi, Tanzania. Blade Messenger delivers speed, security, intelligence, and style in a premium messaging experience.
